### Sales Analysis Dashboard

# Objective
  Analyze sales data to identify trends, top-performing products, and regional performance.

# Tools Used
  Power BI
  Excel
  
# Key Insights
  USA contributes highest number of orders
  Revenue shows strong upward trend
  Low return rate (~1%) indicates product quality

# Features
  KPI Cards (Revenue, Profit, Orders, Return %)
  Revenue Trend Analysis
  Country-wise Performance
  Top Brands Analysis
